Install Eclipse and android development tools :
  Step 1: Install JDK 
    sudo apt-get install openjdk-7-jdk
    
  Step 2: Install Eclipse
    sudo apt-get install eclipse
  
  Step 3: Install JDT i.e Java Development Tools
    sudo apt-get install eclipse-jdt
  
  Step 4: Install Android Development Tools
    Help->Install New Software->Add
      Name : ADT Plugin
      Location : https://dl-ssl.google.com/android/eclipse
      Follow the next instructions.....
      
  Step 5: Restart the eclipse and Install Android SDK
      Mark : Install New SDK
