Xampp Install Process:
Step 1: 
Download : https://www.apachefriends.org/xampp-files/7.3.8/xampp-linux-x64-7.3.8-2-installer.run

Step 2: 
sudo chmod +x xampp-linux-x64-7.3.8-2-installer.run

Step 3:
sudo ./xampp-linux-x64-7.3.8-2-installer.run


Xampp On/Off Commands
Start : 
sudo /opt/lampp/lampp start 

Stop  : 
sudo /opt/lampp/lampp stop
