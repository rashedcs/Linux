sudo chmod +x xampp-linux-x64-7.3.8-2-installer.run

sudo ./xampp-linux-x64-7.3.8-2-installer.run



Xampp Start : sudo /opt/lampp/lampp start 
Xampp Stop  : sudo /opt/lampp/lampp stop


Laravel Install :
curl -sS https://getcomposer.org/installer | php -- --install-dir=/usr/local/bin --filename=composer

##Composer running globally, if we do not give the installation directory
(-- --install-dir=/usr/local/bin --filename=composer)  
then we must move it to the /usr/local/bin directory under 'composer' name.
$ sudo mv composer.phar /usr/local/bin/composer##


Create project : 
composer create-project --prefer-dist laravel/laravel project


The go to your project file as your desired drive and commands : php artisan serve
